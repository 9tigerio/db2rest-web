# Install PostgreSQL

PostgreSQL is currently the most popular relational database. It is available on all leading and upcoming cloud platforms.
However, it is also possible to download and install PostgreSQL and install it on your desktop, laptops and virtual machines.

# Download PostgreSQL

PostgreSQL can be downloaded from multiple sources.

## PostgreSQL Website

PostgreSQL can be downloaded from this [link](https://www.postgresql.org/download/).

## EnterpriseDB Website

Download PostgreSQL can be downloaded from EnterpriseDB website from this [link](https://www.enterprisedb.com/).

## PostgresApp

Another simple and quick option to install and manage multiple versions of PostgreSQL database is to use the postgreApp.
The PostgreApp can be downloaded from this [link](https://postgresapp.com/). Once downloaded follow the instructions
[here](https://postgresapp.com/documentation/install.html) to install the desired version of PostgreSQL.
