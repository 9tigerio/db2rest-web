# Run DB2Rest on DigitalOcean

![Run on DigitalOcean](./assets/run-on-digitalocean.png "Run on DigitalOcean")

It is very easy to run DB2Rest on [DigitalOcean](https://www.digitalocean.com/) cloud platform. It is possible to
run DB2Rest on [DigitalOcean App Platform](https://www.digitalocean.com/products/app-platform) using the Docker image. However, it is also possible to
run DB2Rest on droplets or virtual machines(VM).


[DigitalOcean](https://www.digitalocean.com/) supports different managed databases
    - PostgreSQL
    - MySQL
    - MongoDB
    - Redis
